# Twitter/X Profile Setup Guide

## Quick Setup Checklist

### 1. Profile Picture (PFP)
- **Size:** 400x400 pixels
- **Format:** PNG (transparent background)
- **File:** Use `pfp.png` (generated from `pfp.svg`)
- **Upload:** Settings > Profile > Edit profile > Photo

### 2. Banner Image
- **Size:** 1500x500 pixels
- **Format:** PNG or JPG
- **File:** Use `banner.png` (generated from `logo.svg`)
- **Upload:** Settings > Profile > Edit profile > Banner
- **Tip:** Keep the left side simple (it gets covered by profile info)

### 3. Bio (160 characters max)
```
Curated AI design tools. Hand-picked, tested, honest reviews. New tool every day. 🔗 synthr.online
```
**Character count:** 119/160

**Alternative (if you want shorter):**
```
Daily AI design tool reviews. Curated, tested, honest. 🛠️ synthr.online
```

### 4. Name & Handle
- **Name:** Synthr
- **Handle:** @synthr_tools (or whatever is available)
- **Location:** Leave blank or put "Internet"
- **Website:** https://synthr.online

### 5. Pinned Post

Copy and paste this as your first post, then pin it:

```
Launching Synthr — a curated directory of AI design tools.

Every day, we review one tool so you don't waste time testing 50.

15 tools. Honest reviews. Real pricing. No fluff.

https://synthr.online

Follow for daily AI design tool picks.

#AI #DesignTools #WebDesign
```

**Why this works:**
- Clear value proposition (curated, daily, honest)
- Direct link to site
- Call to action (follow)
- Hashtags for discoverability

## Posting Strategy

### Best Times to Post (EST)
| Day | Best Time |
|-----|-----------|
| Monday | 9-10am |
| Tuesday | 9-11am |
| Wednesday | 9-11am |
| Thursday | 9-10am |
| Friday | 9-10am |
| Weekend | 10-11am |

### Post Types That Work
1. **Tool reviews** (your main content)
2. **Quick tips** (how to use a feature)
3. **Polls** (engagement boost)
4. **Before/after** (visual proof)
5. **Threads** (3-5 tweets for deep dives)

### Engagement Rules
- Reply to ALL comments within 30 minutes
- Quote tweet relevant accounts 2x/day
- Retweet design community posts 3x/day
- Follow 20-30 new accounts/day (designers, AI accounts)
- Unfollow non-followers after 3 days

### Growth Milestones
| Followers | Action |
|-----------|--------|
| 0-100 | Post daily, engage heavily |
| 100-500 | Start threads, tag tools |
| 500-1K | Add polls, run a giveaway |
| 1K+ | Apply for affiliate programs |

## Tools to Use
- **Schedule:** TweetDeck (free) or Typefully (paid)
- **Analytics:** Twitter Analytics (built-in)
- **Images:** Canva (ironic but works) for quick graphics
- **Link tracking:** Use UTM parameters (synthr.online?utm_source=twitter)

---

## Red Flags (Don't Do These)
- Don't post more than 3x/day (looks spammy)
- Don't use more than 3 hashtags per post
- Don't buy followers (kills reach)
- Don't only post links (add value first)
- Don't ignore replies (kills engagement)
